S.A.R.G.I.S - как запускать

1. Главный файл бота:
   sargis_bot.py

2. Самый простой запуск:
   дважды нажми run_bot.bat

3. Mini App:
   дважды нажми run_mini_app.bat
   потом открой в браузере:
   http://127.0.0.1:8000

4. Какие файлы сейчас основные:
   sargis_bot.py
   mini_app.py
   requirements.txt
   run_bot.bat
   run_mini_app.bat
   папка sargis
   папка templates
   папка static

5. Старые файлы перенесены в папку legacy.
   Они больше не нужны для нового запуска.

6. Если бот не стартует:
   сначала установи зависимости командой:
   pip install -r requirements.txt
